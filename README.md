# Управление движением денежных средств (ДДС)

Веб-приложение для учета, управления и анализа поступлений и списаний денежных средств.

## Функциональность

- Создание, редактирование, удаление записей ДДС
- Фильтрация по дате, статусу, типу, категории, подкатегории
- Управление справочниками (статусы, типы, категории, подкатегории)
- Автоматическая валидация на клиенте и сервере
- Интуитивно понятный интерфейс на Bootstrap

## Технологии

- Python 3.8+
- Django 4.0+
- SQLite
- Bootstrap 5
- jQuery

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd cashflow_project
```

### 2. Создание виртуального окружения

```bash
# для Linux/Mac
python -m venv venv
source venv/bin/activate
 # для Windows
venv\Scripts\activate 
```

### 3. Установка зависимостей

```bash
pip install django
```

### 4. Создание файла requirements.txt

```bash
pip install django
```

### 5. Настройка базы данных

```bash
pip install django
```

### 6. Создание суперпользователя (для доступа к админке)

```bash
python manage.py createsuperuser
```

### 7. Запуск сервера

```bash
python manage.py createsuperuser
```
Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/
